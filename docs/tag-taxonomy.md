# Rozaya Boutique Tag Taxonomy

## Rules

- Use all lowercase tags.
- Use hyphens for spaces.
- Use the `key:value` namespaced prefix pattern for filterable dimensions.
- Use plain, unprefixed tags for category.
- Every product should include the universal `jewelry` tag plus at least one category tag.

## Category

Category tags do not use a prefix:

- `ring`
- `necklace`
- `bracelet`
- `earrings`
- `brooch`
- `charm`
- `pendant`
- `cufflinks`
- `watch`
- `pin`
- `bangle`
- `chain`

## Metal

Use the `metal:` prefix:

- `metal:14k-gold`
- `metal:18k-gold`
- `metal:24k-gold`
- `metal:white-gold`
- `metal:rose-gold`
- `metal:yellow-gold`
- `metal:platinum`
- `metal:sterling-silver`
- `metal:silver`
- `metal:brass`
- `metal:copper`
- `metal:gold-filled`
- `metal:gold-plated`

## Stone

Use the `stone:` prefix:

- `stone:diamond`
- `stone:sapphire`
- `stone:emerald`
- `stone:ruby`
- `stone:pearl`
- `stone:opal`
- `stone:topaz`
- `stone:amethyst`
- `stone:garnet`
- `stone:aquamarine`
- `stone:turquoise`
- `stone:onyx`
- `stone:jade`
- `stone:citrine`
- `stone:tourmaline`
- `stone:no-stone`

## Era

Use the `era:` prefix:

- `era:georgian` (1714–1837)
- `era:victorian` (1837–1901)
- `era:art-nouveau` (1890–1910)
- `era:edwardian` (1901–1910)
- `era:art-deco` (1920–1935)
- `era:retro` (1935–1950)
- `era:mid-century` (1950–1970)
- `era:vintage` (1970–2000)
- `era:contemporary` (2000+)

## Style

Use the `style:` prefix:

- `style:antique` (100+ yrs)
- `style:estate`
- `style:signed`
- `style:artisan`
- `style:collectible`

## Condition

Use the `condition:` prefix:

- `condition:excellent`
- `condition:very-good`
- `condition:good`
- `condition:fair`

## Universal Tags

Every product should include:

- `jewelry`
- At least one category tag, such as `ring`, `necklace`, or `brooch`

## Examples

### Art Deco Diamond Ring

`jewelry, ring, metal:platinum, stone:diamond, era:art-deco, style:estate, condition:excellent`

### Victorian Ruby Brooch

`jewelry, brooch, metal:14k-gold, stone:ruby, era:victorian, style:antique, condition:very-good`

### Contemporary Artisan Silver Bracelet

`jewelry, bracelet, metal:sterling-silver, stone:no-stone, era:contemporary, style:artisan, condition:excellent`

### Mid-Century Pearl Necklace

`jewelry, necklace, metal:14k-gold, stone:pearl, era:mid-century, style:estate, condition:very-good`

### Signed Tiffany Diamond Pendant

`jewelry, pendant, metal:platinum, stone:diamond, era:contemporary, style:signed, condition:excellent`
